### Guest
| Kategori                            | Deskripsi                                                                                                                                                                       |
| ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Autentikasi & Profil**            | - Register / Sign up <br> - Login / Logout <br> - Melihat & memperbarui profil (nama, email, password, telepon) <br> - Reset password                                           |
| **Hotel Browsing & Searching**      | - Melihat daftar hotel <br> - Mencari hotel berdasarkan kota, tanggal, harga, rating, fasilitas <br> - Melihat detail hotel (foto, deskripsi, kamar, fasilitas, review, lokasi) |
| **Booking / Pemesanan**             | - Memilih tipe kamar <br> - Menentukan tanggal check-in dan check-out <br> - Melakukan booking <br> - Melihat ringkasan harga <br> - Melakukan pembayaran                       |
| **Payment & Invoice**               | - Melihat status pembayaran <br> - Melihat & mengunduh invoice <br> - Membatalkan pesanan sebelum batas waktu                                                                   |
| **Review & Feedback**               | - Memberikan rating & review hotel setelah menginap                                                                                                                             |
| **Membership / Loyalty (opsional)** | - Mendapat poin / reward <br> - Melihat level keanggotaan (Regular, Silver, Gold, Platinum)                                                                                     |
| **Notifikasi**                      | - Mendapatkan notifikasi konfirmasi booking, pengingat check-in, promo, dll                                                                                                     |


Admin
| Kategori                             | Deskripsi                                                                                                                                                               |
| ------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Autentikasi & Role**               | - Login admin (biasanya dengan level role: Super Admin, Hotel Manager, Frontdesk Staff) <br> - Manajemen role & permissions                                             |
| **Hotel Management**                 | - Menambah / mengedit / menghapus hotel <br> - Menambah / mengatur tipe kamar <br> - Menentukan kapasitas, harga, fasilitas, dan stok kamar                             |
| **Booking Management**               | - Melihat semua booking dari guest <br> - Mengonfirmasi atau membatalkan booking <br> - Melihat status check-in/check-out tamu <br> - Mengatur refund (jika dibatalkan) |
| **User Management (Guest)**          | - Melihat daftar tamu <br> - Menonaktifkan akun yang bermasalah (spam, fraud)                                                                                           |
| **Payment & Transaction Management** | - Melihat riwayat pembayaran <br> - Verifikasi pembayaran manual (jika offline)                                                                                         |
| **Report & Analytics**               | - Melihat statistik booking per periode <br> - Pendapatan harian / bulanan <br> - Laporan okupansi kamar <br> - Feedback / rating performa hotel                        |
| **Content Management**               | - Upload foto hotel / kamar <br> - Kelola deskripsi, fasilitas, dan kebijakan (policy)                                                                                  |
| **Promotions & Discounts**           | - Membuat dan mengelola promo, voucher, atau potongan harga                                                                                                             |
| **System Configuration**             | - Mengelola setting umum (timezone, mata uang, language) <br> - Backup database (opsional)                                                                              |
